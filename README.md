# NLP Assignment Project

This repository contains a comprehensive NLP application covering:

1. Social Media Monitoring Dashboard (Assignment 1)
2. Multilingual Sentiment Analysis Research (Assignment 2)
3. Model Performance Benchmarking (Assignment 3)

## Setup

```bash
pip install -r requirements.txt
```

## Run Demo

```bash
python run_demo.py
```

## Folder Structure

- `social_monitor.py`: Social media monitoring dashboard
- `multilingual_research.py`: Multilingual NLP research
- `model_benchmark.py`: Model performance benchmarking
- `docs/API_reference.md`: Technical API documentation
